# 🎴 Random Card Picker

A small web project that draws a random hand of playing cards. Choose between 1 and 10 cards, add 0, 1, or 2 jokers, and enjoy a colorful card reveal.

## ▶ Play Live Demo

[Click Here to Draw Cards](https://rajkarthik2003.github.io/cards)

## Features
- Draw 1–10 unique cards from a full deck
- Include 0, 1, or 2 jokers
- Animated and responsive card UI
- Suit colors for red and black cards
- Live status showing deck size and drawn count

## How to use
1. Open `index.html` in your browser.
2. Choose how many cards to draw.
3. Select how many jokers to include.
4. Press **Draw cards** to refresh your hand.

## Notes
- Diamonds and hearts are shown in red.
- Spades and clubs are shown in black.
- Jokers appear as special visual cards.

## Development
- `index.html` — page layout
- `styles.css` — UI styling
- `script.js` — card generation and draw logic
